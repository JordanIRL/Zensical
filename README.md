# Jordan's KB

A personal knowledge base.

Live site: [kb.jordanflynn.com](https://kb.jordanflynn.ie)

## Local development

```sh
pip install -r requirements.txt
zensical serve
```

Content lives in [`docs/`](docs/); site config is in [`zensical.toml`](zensical.toml).
